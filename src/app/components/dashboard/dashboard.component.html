המערכת פותחה כפתרון גמיש ורחב־היקף לחישוב נוסחאות דינמיות על כמויות נתונים גדולות.

היא מאפשרת להגדיר נוסחאות בזמן ריצה, להריץ אותן על מאגר נתונים רחב, ולהשוות בין מספר מנועי חישוב שונים — וכל זאת ללא צורך בשינוי קוד בכל פעם שנוסחה משתנה.

⚙️ ארכיטקטורה כללית

הליבה של המערכת היא מנגנון הרצה מרכזי שמנהל את כל תהליך החישוב:

  • 📋 טעינת הנוסחאות ממסד הנתונים
  • ⚡ הפעלת כל מנועי החישוב במקביל
  • 📦 עיבוד הנתונים במנות (Paging)
  • 💾 שמירת תוצאות החישוב
  • 🕐 תיעוד זמני הריצה והשוואתם

כל מנוע חישוב ממומש כסרוויס עצמאי שמיישם ממשק אחיד (IMethodEngine).
הגישה הזו מאפשרת:

הוספת מנועים חדשים בקלות
השוואת ביצועים בין שיטות שונות
תחזוקה פשוטה והפרדה ברורה בין לוגיקות

🔬 חישוב במערכת

המערכת כוללת שלושה מנועי חישוב שונים, שכל אחד מייצג גישה אחרת לפתרון נוסחאות דינמיות:

1 SQL SQL Dynamic

הנוסחה מתורגמת ל־SQL דינמי ומורצת ישירות על מסד הנתונים.

  • ⚡ מהיר במיוחד על טבלאות גדולות
  • 🎯 החישוב מתבצע קרוב לנתונים
  • ✅ תומך בתנאים ובפעולות מתמטיות
2 C# Roslyn C#

הנוסחה מקומפלת בזמן ריצה לקוד C# ומורצת כפונקציה לכל דבר.

  • 🔧 גמישות גבוהה
  • ⚡ ביצועים מצוינים לאחר קומפילציה
  • 🧠 מתאים לנוסחאות מורכבות
3 AST DynamicExpresso (AST Interpreter)

מפרש את הנוסחה לעץ ביטויים ומחשב אותה ללא קומפילציה מלאה.

  • ➕ קל להרחבה
  • ✅ גמישות גבוהה
  • 🔧 מאפשר הוספת פונקציות מותאמות
🧩 סיכום
